# docs

`joy7758/docs` 仓库的稀疏本地工作区。

这个检出只保留文档配置和当前选中的可编辑源文件，不会把完整上游目录树全部拉入当前工作区。

## 这里有什么

- `AGENTS.md` 记录了这个工作区的文档编辑规则。
- `src/docs.json` 包含 Mintlify 站点配置。
- `reference/` 预留给生成式参考文档产物。

## 这个工作区是用来做什么的

- 审查文档结构
- 在稀疏检出已包含的范围内编辑人工维护的文档面
- 在修改文档前检查站点配置

## 这个工作区不是用来做什么的

- 它不是完整的上游文档仓库检出
- 它不是生成后的发布产物
- 它不是直接编辑自动生成参考内容的地方
